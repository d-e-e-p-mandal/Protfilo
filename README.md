Portfolio: https://deep-mandal-portfolio.netlify.app/

---

## 📌 About the Project
**Protfilo** is a personal portfolio website built to represent my profile as a software developer.  
It highlights my projects, technical skills, and background using modern frontend technologies with smooth scrolling and animation effects.

This project is designed to be **fast, responsive, and visually engaging**, suitable for recruiters and collaborators to explore my work.

---

## ✨ Features
- Responsive design (mobile, tablet, desktop)
- Smooth scrolling experience
- Modern UI with animations
- Projects showcase section
- Clean and scalable folder structure
- Fast deployment with Netlify

---

## 🛠️ Tech Stack
- **Frontend:** React / Vite  
- **Styling:** CSS / Tailwind CSS  
- **Animations:** GSAP, Lenis  
- **Version Control:** Git & GitHub  
- **Deployment:** Netlify
